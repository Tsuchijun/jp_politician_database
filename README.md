# 政治家役職要覧 (Japanese Politician Database)

日本の歴代大臣・党役員の在任期間を可視化したWebアプリケーションです。
明治時代から現代までの、内閣・国会・政党・地方自治体の主要役職の変遷をタイムライン形式で閲覧できます。

## 公開サイト
👉 **[こちらから閲覧できます](https://tsuchijun.github.io/jp_politician_database/)**

## 主な機能
* **タイムライン表示:** 歴代の役職者の在任期間をガントチャート形式で表示
* **検索機能:** 政治家の氏名で検索し、詳細経歴を表示
* **フィルタリング:** 内閣、国会、政党、地方自治体ごとの表示切り替え
* **ランキング:** 在任日数や就任回数のランキングを表示
